Example Useage

A simple example of retrieving the Country, State, City in JSON format from the Ziptastic! API.

 

Code For Example Above:

$(function() {
	$("#zippy-do-da").click(function(event) {
		//Get the entered Zipcode and prevent the buttons default action
	  	var zipcodeEntered = $('#ziput').val();
	  	event.preventDefault();
	  	//Make an AJAX request to the Ziptastic! api.
	  	$.get(
	  		'http://zip.elevenbasetwo.com?zip=' + zipcodeEntered,
			function(apiData) {
				//Update our output element with the data returned from the api
		  		$('#zippdy-day').text('City: ' + apiData.city + ', State: ' + apiData.state + '. Country: ' + apiData.country)
        	}
		);
	});
});
